## Style

You can customize how your Roblox bootstrapper should look. Fishstrap by default comes installed with some preset bootstrapper themes for you to select. 

You can select your own bootstrapper themes by clicking `Custom` style.

<img src="/docs_appearance_presets.webp" alt="preset_bootstrappers_preview" />

## Icon
Choose what Icon Fishstrap should use for bootstrappers by default. Select which icon you want via the dropmenu.

<img src="/docs_appearance_dropmenu.webp" alt="dropmenu" />

## Bootstrapper customization 

<Aside type="note">Some custom bootstrappers might override the current icon or text with their own.</Aside>

### Title
The text that shows itself as the title of the bootstrapper.

### Custom Icon
Custom icon to set beside the title. Set Icon as `Custom` to use it.
